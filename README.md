# AfroHeat Fitness

AfroHeat is a women-only fitness center celebrating African culture through dance, strength training, and community events.

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

## Build and Preview

- Build: `npm run build`
- Preview: `npm run preview`

## Vercel Deployment

1. Install Vercel CLI: `npm i -g vercel`
2. Login: `vercel login`
3. Deploy: `vercel`

The site is configured for static hosting with SPA routing via vercel.json.
